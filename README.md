# Termcaster

A simple terminal-based raycaster written in C using [Termbox2](https://github.com/termbox/termbox2).

## Demo

[Link](termcaster.mp4)

## Build & Run

```bash
make
./termcaster
```

### Controls
- Arrow keys: Move and rotate
- ESC: Quit
